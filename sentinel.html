Autonomous Mobile Robot

A ROS2-based robot capable of SLAM and autonomous navigation in warehouse environments.

Role

Lead Engineer

Date

Aug 2024 - Dec 2024

Tech Stack

Python, C++, ROS2, Gazebo

Hardware

Raspberry Pi 4, LiDAR, Arduino

The Challenge

Explain the problem you were trying to solve here. For example: "Traditional warehouse robots require expensive magnetic tracks. The goal was to create a cost-effective robot that navigates using natural features."

Technical Implementation

Here you describe the "Mechatronics" magic. Talk about the integration between the mechanical design and the software.

The control loop was implemented using a PID controller on the Arduino, communicating with the Raspberry Pi via serial. The SLAM algorithm utilized was Gmapping for static mapping and AMCL for localization.

Results

The robot successfully navigated a 500sq ft environment, avoiding dynamic obstacles with a success rate of 95%.
